Terminologia basica de HTML Browser:
  • Un browser por sus siglas en ingles es un software, o programa que permite el accesso a internet, descifrando la informacion de diferentes tipos de archivos y para que estos logren ser visualizados y logre alcanzar conexiones de una pagina a otra instalada en cualquier computador conectado a internet, a esto se le llama navegacion, del cual se origina el calificativo navegador
    • Hipermedia:
  • El termino hipermedia sirve para designar el conjunto de metodos o procedimientos para escribir, diseñar o componer contenidos que integren soportes como: texto, imagen, video, audio, mapas y otros soportes de informacion emergentes, de tal modo que el resultado obtenido, ademas tenga la posibilidad de interactuar con los usuarios
    • URL:
  • URL significa Uniform Resource Locator (Localizador de Recursos Uniforme). Una URL no es mas que una direccion que es dada de un recurso unico en la web. En Teoria, cada URL valida apunta a un unico recurso, dichos recursos pueden ser paginas HTML, documentos CSS, imagenes, etc.
    • Etiquetas:
  • Las etiquetas HTML son indicadores del lenguaje de marcado de hipertexto que le dan formato y estructura al contenido de las paginas web. Es un estandar adoptado por todos los desarrolladores que, mediante etiquetas de texto, le permite a los navegadores interpretar los elementos de una pagina web, no importa donde, cuando o en que idioma fue creado, para mostrar su contenido a los usuarios
    • HTML:
  • El lenguaje de marcado de hipertexto es el codigo que se utiliza para estructurar y desplegar una pagina web y sus contenidos. Por ejemplo, sus contenidos podran ser parrafos, una lista con viñetas o imagenes y tablas de datos, Como lo sugiere el titulo, este articulo te dara conocimiento basico de HTML y su funcion
